Preguntas

1. ¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es la red global que conecta computadoras en todo el mundo, mientras que la World Wide Web es un servicio que funciona sobre Internet y permite acceder a páginas web mediante navegadores.

2. ¿Cuáles son las partes de un URL?

Un URL está formado por varias partes: el protocolo (como https), el dominio (como google.com) y la ruta que indica la ubicación específica dentro del sitio.

3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Los métodos HTTP indican qué acción se quiere realizar en un servidor: GET obtener información, HEAD obtener información sin contenido, POST enviar datos, PUT actualizar información completa, PATCH actualizar una parte y DELETE eliminar información.

4. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML? ¿Por qué?

Se utiliza el método POST, porque los datos se envían en el cuerpo de la solicitud y no en la URL, lo que hace que no sean visibles directamente.

5. ¿Qué método HTTP se utiliza cuando accedes a una página con un URL?

Se utiliza el método GET, ya que el navegador solicita la información al servidor para mostrar la página.

6. Un servidor web devuelve código 200, ¿qué significa? ¿Hubo error?

El código 200 significa que la solicitud fue exitosa y todo funcionó correctamente. No ocurrió ningún error.

7. ¿Es responsabilidad del desarrollador corregir un error 404? ¿Por qué?

Sí, porque indica que la página no fue encontrada. Puede ser por enlaces incorrectos o páginas eliminadas, y el desarrollador debe asegurarse de que los enlaces funcionen correctamente.

8. ¿Es responsabilidad del desarrollador corregir un error 500? ¿Por qué?

Sí, porque indica un error interno en el servidor. Esto significa que hay un problema en el sistema o en el código que debe ser corregido.

9. ¿Qué significa que un atributo HTML esté deprecated? Menciona ejemplos

Significa que ese atributo ya no se recomienda usar porque fue reemplazado por nuevas formas en HTML5. Ejemplos son font, center y big.

10. ¿Qué componentes de estructura y estilo tiene una tabla?

Una tabla en HTML tiene table para la tabla completa, tr para filas, td para celdas y th para encabezados.

11. ¿Cuáles son los principales controles de una forma HTML5?

Los principales controles son input, textarea, button y select.

12. ¿Cuál es el ciclo de vida y desarrollo de los sistemas de información?

El ciclo de vida es el proceso completo desde que se crea un sistema hasta que deja de usarse. El ciclo de desarrollo incluye análisis, diseño, desarrollo, pruebas, implementación y mantenimiento.
