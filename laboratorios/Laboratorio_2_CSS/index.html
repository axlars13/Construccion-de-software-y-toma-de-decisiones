Reflexión

¿Cuál es la diferencia entre padding y margin?

El padding es el espacio interno dentro de un elemento, mientras que el margin es el espacio externo que separa un elemento de otros.

¿Por qué se recomienda usar box-sizing: border-box?

Porque hace más fácil controlar el tamaño de los elementos, ya que el padding y el borde se incluyen dentro del tamaño total.

¿Qué significa diseñar con enfoque mobile-first?

Significa que primero se diseña la página para dispositivos móviles y después se adapta a pantallas más grandes.

¿En qué situaciones preferirías rem sobre px?

Se usa rem cuando se quiere que el tamaño sea adaptable según la pantalla o configuración del usuario.
